Modal Sederhana

Ini adalah contoh modal responsif menggunakan Tailwind CSS v4. Modal ini akan menyesuaikan ukuran berdasarkan layar perangkat.

Coba resize browser Anda untuk melihat responsifitas modal.

Fitur Responsif

Modal ini menggunakan grid system Tailwind CSS yang responsif. Pada layar kecil, konten akan ditampilkan dalam satu kolom.

Mobile First

Dirancang dengan pendekatan mobile-first. Coba buka di perangkat mobile untuk pengalaman terbaik.

Ukuran Fleksibel

Modal dapat diatur dengan berbagai ukuran maksimum dan tinggi sesuai kebutuhan.

Scroll Otomatis

Jika konten terlalu panjang, modal akan menampilkan scrollbar secara otomatis.

Contoh Konten Panjang

Tailwind CSS v4.1.13 menyediakan utility classes yang memudahkan pembuatan komponen responsif seperti modal ini.

Dengan menggunakan breakpoints seperti sm:, md:, lg:, kita dapat mengontrol tampilan di berbagai ukuran layar.

Modal ini menggunakan fixed positioning untuk tetap berada di tengah layar meski ketika di-scroll.
